# Şarap Kalite Analizi <br/>
### Proje Amacı <br/>

Projemde şarap kalitesini belirleyen özellikler incelenip sınıflandırılmıştır. <br/>
Kullandığım veri seti; https://www.kaggle.com/datasets/nareshbhat/wine-quality-binary-classification/code <br/>
Veri setimde 1599 adet örneklem 11 adet özellik vardır ve çıktı olarak iyi veya kötü çıktıları verecektir. <br/>
Projemde KNN, Logistic Regresyon, Random Forest Sınıflandırma yöntemleri kullanılmıştır.
